# ToDoリスト

シンプルなタスク管理アプリ。Next.js + Tailwind CSS で構築。

## 機能

- タスクの追加（Enterキーまたは「追加」ボタン）
- 完了チェック（チェックマークをクリック）
- タスクの削除（ゴミ箱アイコンをホバーして削除）
- フィルター（すべて / 未完了 / 完了済み）
- データはブラウザの localStorage に保存

## 開発

```bash
npm install
npm run dev
```

## Vercel デプロイ

1. GitHubにリポジトリを作成してプッシュ
2. [vercel.com](https://vercel.com) でNew Projectを選択
3. リポジトリを選択してDeployをクリック

設定は不要で自動でデプロイされます。
